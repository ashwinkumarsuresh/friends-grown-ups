# Quick Start Guide

## Friends Game - Grown-Ups Edition (18+)

### Step 1: Get an API Key

Choose one of these providers:

**Google Gemini (FREE)** - Recommended
- Go to: https://aistudio.google.com/app/apikey
- Create an API key (starts with `AIza`)

**OpenAI GPT** (Paid)
- Go to: https://platform.openai.com/api-keys
- Create a key (starts with `sk-`)

**Anthropic Claude** (Paid)
- Go to: https://console.anthropic.com/
- Create a key (starts with `sk-ant-`)

### Step 2: Start the Server

```bash
cd /Users/ashwin/Documents/Code/Game/friends-grown-ups
python3 proxy-server.py
```

### Step 3: Open in Browser

Go to: **http://localhost:3002**

### Step 4: Play!

1. Add at least 2 friends
2. Select your AI provider
3. Enter your API key
4. Click "Start Game!"
5. Choose your spice level (Mild, Medium, Spicy, or Intimate)
6. Pick a topic and answer the question!
